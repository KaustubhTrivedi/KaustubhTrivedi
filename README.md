### 🤖 [Rendure](https://rendure.kaustubhsstuff.com)

[![Live](https://img.shields.io/badge/Live-rendure.kaustubhsstuff.com-A9FEF7?style=flat-square)](https://rendure.kaustubhsstuff.com)
[![Stack](https://img.shields.io/badge/Stack-Next.js%20%7C%20FastAPI%20%7C%20LangChain%20%7C%20Celery-4ECDC4?style=flat-square)](#)

**Agentic Resume Tailoring Pipeline**

A full-stack AI platform that automates resume tailoring for specific job descriptions. Built with a multi-agent LangChain pipeline backed by Gemini Flash Lite, it crawls and parses JDs, rewrites YAML-based RenderCV resumes for ATS optimisation, and renders them to PDF via the RenderCV CLI — all orchestrated asynchronously through Celery and delivered in a clean Next.js UI.

#### ⚙️ How it works:
* 🔍 **JD Ingestion**: JinaAI-powered scraping extracts structured requirements from any job posting URL
* 🧠 **Multi-Agent Rewrite**: LangChain agents rewrite resume YAML to align with the target role
* 📄 **PDF Rendering**: RenderCV CLI compiles the tailored YAML into a polished, ATS-ready PDF
* 🚀 **Async Processing**: Celery + Redis task queue handles rendering pipelines non-blocking

### 🦅 OpenClaw / HARVESTER + SCOUT

[![Stack](https://img.shields.io/badge/Stack-OpenClaw%20%7C%20Python%20%7C%20Discord-5865F2?style=flat-square)](#)
[![Homelab](https://img.shields.io/badge/Self--Hosted-Homelab%20Deployed-4ECDC4?style=flat-square)](#)

**Automated Job Discovery & Alert System**

A multi-agent job intelligence system built on OpenClaw, running self-hosted on my homelab. HARVESTER crawls Irish employer career pages on a schedule, surfacing new software engineering roles matching my criteria. SCOUT processes incoming job alert emails, parses structured data from them, and routes curated digests to Discord — keeping my job pipeline populated without manual trawling.

#### 🎯 What it does:
* 🕷️ **HARVESTER**: Scheduled crawler that monitors career pages across Irish tech employers
* 📬 **SCOUT**: Email-triggered agent that parses job alerts and extracts structured role data
* 📢 **Discord Delivery**: Formatted job digests routed to a private Discord server for review
* 🏠 **Self-Hosted**: Fully deployed on homelab infrastructure via Docker Compose

### ☤ Hermes Agent Deployment

[![Stack](https://img.shields.io/badge/Stack-Hermes%20%7C%20Python%20%7C%20Telegram-229ED9?style=flat-square)](#)
[![Orchestrator](https://img.shields.io/badge/Role-OpenClaw%20Orchestrator-A9FEF7?style=flat-square)](#)

**Self-Improving AI Agent — Managing My OpenClaw Infrastructure**

Running a self-hosted instance of [Hermes Agent](https://github.com/NousResearch/hermes-agent) (by Nous Research) as the orchestration and control layer over my OpenClaw setup. Hermes provides persistent memory, a built-in learning loop, and a Telegram gateway — letting me manage, trigger, and monitor my HARVESTER and SCOUT agents from my phone, anywhere. Skills built during use are retained across sessions, making the agent progressively more aware of my job search context over time.

#### 🔧 Setup highlights:
* 📱 **Telegram Gateway**: Control OpenClaw agents remotely via natural language over Telegram
* 🧠 **Persistent Memory**: Cross-session recall keeps Hermes aware of ongoing job search context
* 🔄 **Skill Accumulation**: Agents learn and improve from repeated job-pipeline interactions
* 🏠 **Homelab Deployed**: Self-hosted alongside the rest of my stack via Docker
